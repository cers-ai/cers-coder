## 🎯 使用方式

### 快速开始
```bash
# 1. 克隆项目
git clone <repository-url>
cd cers-coder

# 2. 安装依赖
pip install -r requirements.txt
pip install -e .

# 3. 准备需求文件
cp 0.request.md.example 0.request.md
# 编辑 0.request.md 描述你的项目需求

# 4. 启动开发
cers-coder start
```

### Docker部署
```bash
# 简化版部署
docker-compose -f docker-compose.simple.yaml up -d

# 完整版部署
docker-compose up -d
```
